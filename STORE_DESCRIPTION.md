# Chrome Web Store 掲載情報

## 短い説明

claude.aiでEnterを改行、Shift+Enter（またはCmd+Enter）を送信に変更。送信時にGoogle検索を同時実行する機能付き。

## 詳細な説明

claude.aiのチャット入力でEnterキーとShift+Enterキーの動作を入れ替えるChrome拡張です。

機能:
- 送信キーを Shift+Enter または Cmd+Enter から選択可能（ポップアップから設定）
- Enterキーで改行を挿入
- Escキーでチャット入力欄にフォーカス
- メッセージ編集フォームにも対応
- IME入力中の誤送信を防止
- 送信時にGoogle検索を同時実行（ON/OFF切替可能）
  - 入力テキストの先頭N文字をクエリとして自動検索
  - 検索文字数はカスタマイズ可能（10〜500文字）
  - 同じタブを再利用するのでタブが増えない

設定はブラウザに同期保存され、リロード不要でリアルタイムに反映されます。

対象サイト: claude.ai
